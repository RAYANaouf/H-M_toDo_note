### Overview
**H@W Note** is a custom application designed to help you organize your notes effectively. It offers robust features tailored to optimize user experience:

- **Versatile Note Formatting:** Create notes with paragraphs, checkboxes (e.g., for to-do lists), and images.
- **Customization Options:** Distinguish your notes using categories and colors to stay organized.
- **Enhanced Security:** Optionally secure individual notes with passwords to protect sensitive information.
- **App-Level Security:** Set a master password to safeguard all your notes.
